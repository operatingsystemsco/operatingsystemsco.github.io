Change Number:  U014/1
On the dd manual page (Page 251), replace the second paragraph of
the cbs= description with the following text and mark it
as Extended behaviour:
  "This operand must also be specified if the "conv=" operand is specified
  with a value of "ascii", "ebcdic" or "ibm". For a "conv=" operand with an
  "ascii" value, the input is handled as described for the "unblock" value
  except that characters are converted to ASCII before any trailing space
  characters are deleted. For "conv=" operands with "ebcdic" or "ibm" values,
  the input is handled as described for the "block" value except that the
  characters are converted to EBCDIC or IBM EBCDIC, respectively, after any
  trailing space characters are added."
