Change number : XMPTN-1.1

Title

Effect of addition of OSI Transport User Characteristics on specification of SNA Transport Provider Characteristics.

Qualifier

Major Technical

Rationale

The compensations required for an OSI Transport User over the SNA Transport Provider need to be added to the table of compensations required, as given in Chapter 11 (SNA Transport Provider Characteristics) of the XMPTN Access Node specification.

Change

Add a new column to Table 11-4, Pages 138-139.
This column contains the compensations when an OSI Transport User runs over an SNA Transport Provider.

(1) The heading of column should be "OSI"

(2) Put "Yes" in this new column for the following compensations:
NOCOMP:
DA_TERM:
RECSEG:

(3) Put "Note 1" in this column for the "EXP:" compensation.

Change number : XMPTN-1.2

Title

Effect of addition of SPX/IPX Transport Provider Characteristics on specification of AF_INET, SNA and NetBEUI Transport User Characteristics.

Qualifier

Major Technical

Rationale

The compensations required for support of AF/INET, SNA and NetBEUI Transport Users run over an SPX/IPX Transport Provider need to be added to the relevant tables of compensations required, as given in Chapters 8 (AF_INET Transport User), 10 (SNA Transport User) and 12 (NetBEUI Transport User) of the XMPTN Access Node specification.

Change

(1) In Chapter 8, add a new column to Table 8-6. This column contains the compensations when an AF_INET Transport User runs over an SPX/IPX Transport Provider.

  • The heading of column should be "SPX/IPX"
  • Put "Yes" in this new column for the following compensations:
    NOCOMP:
    EXP:
    SO_TERM:

(2) In Chapter 10, add a new column to Table 10-5.
This column contains the compensations when an SNA Transport User runs over an SPX/IPX Transport Provider.

  • The heading of column should be "SPX/IPX"

  • Put "Yes" in this new column for the following compensations:
    NOCOMP:
    EXP:
    EXP_ACK:
    DA_TERM:
    REC_SEG:
    DG_OOB_DATA

(3) In Chapter 12, add a new column to Table 12-4.
This column contains the compensations when an NetBEUI Transport User runs over an SPX/IPX Transport Provider.

  • Heading of column should be "SPX/IPX"

  • Put "Yes" in this new column for the following compensations:
    NOCOMP:
    REC_SEG:
