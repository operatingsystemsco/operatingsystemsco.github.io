NAME

fgetpos - get current file position information
Note:
The XSH specification for fgetpos() contains the basic definition of this interface. The following additional information pertains to Sockets.

ERRORS

The fgetpos() function may fail if:
[ESPIPE]
The file descriptor underlying stream is associated with a socket.
