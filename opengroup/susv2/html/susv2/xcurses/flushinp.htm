 NAME

flushinp - discard input

 SYNOPSIS



#include <curses.h>

int flushinp(void);

 DESCRIPTION

The flushinp() function discards (flushes) any characters in the input buffer associated with the current screen.

 RETURN VALUE

The flushinp() function always returns OK.

 ERRORS

No errors are defined.
