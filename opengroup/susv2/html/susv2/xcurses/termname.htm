 NAME

termname - get terminal name

 SYNOPSIS



#include <curses.h>

char *termname(void);

 DESCRIPTION

The termname() function obtains the terminal name as recorded by setupterm().

 RETURN VALUE

The termname() function returns a pointer to the terminal name.

 ERRORS

No errors are defined.
