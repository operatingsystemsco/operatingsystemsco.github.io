 NAME

creat - create a new file or rewrite an existing one

 SYNOPSIS



#include <sys/types.h>
#include <sys/stat.h>
#include <fcntl.h>

int creat(const char *path, mode_t mode);

 DESCRIPTION

The function call: 

creat(path, mode)
is equivalent to: 

open(path, O_WRONLY|O_CREAT|O_TRUNC, mode)

 RETURN VALUE

Refer to open().

 ERRORS

Refer to open().

 EXAMPLES

None.

 APPLICATION USAGE

None.

None.
