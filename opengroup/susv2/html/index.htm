The Single UNIX® Specification, Version 2 (HTML)

Two versions are available:

The CD-ROM Version

CD-ROM Version: This uses short filenames and can be viewed directly off the CD-ROM.

Archive Version

An alternate html version using long filenames within the html, can be installed on the local hard disk. The System Requirements are: 12 MB disk space free and support for long filenames (i.e. UNIX, OS/2, Windows 95, Windows NT, Macintosh)

To download your selection, you may need to use the "Save Next Link As..." or "Save to Disk" functions of your Web browser.

Available Formats:

Tar format
Compress'd tar format
ZIP format

Included on the CD-ROM are some sample cgi scripts to dynamically search the long filename version of the html specification. These files are in the cgi.tar tar archive. These have been successfully used on several UNIX operating systems. These are provided "as is", they are not intended for commercial use, see each individual file for the copyright notice.
