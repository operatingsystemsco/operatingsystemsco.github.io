Made with Macromedia Trademark ArtworkThe artwork for the Made with Macromedia logo is contained in three files on this disk. These files are to be used in accordance with the Made with Macromedia Run-time Distribution Agreement. Please refer to the Logo Usage Guidelines located in the MWMDOCS folder for correct placement instructions.For Print Usage:MWMLOGO.EPS is an EPS file for use in placing the Made with Macromedia logo in printed material such as packaging and manuals.For Screen Usage:MWMLOGO.PICT is a file for use within Director productions.For information regarding Made with Macromedia for Shockwave please go to the following URL: http://www.macromedia.com/support/mwmp/shockwave/Additional information can be obtained from our web site: www.macromedia.com.
